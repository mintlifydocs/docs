---
title: Create an API Account
---
Before you can make your first request, you need a Lalals API Account.

1. Head to [lalals.com/api](https://lalals.com/api) and click on **Get started for free** or select one of the available API Packages:
    - **Free** 🐣
    - **API Basic** 🐆
    - **API Pro** 🗿 (best pricing per generated song)

2. Enter your payment information.

3. 🎉 Congratulations! You now have a Lalals API Account and are ready to start making requests.
