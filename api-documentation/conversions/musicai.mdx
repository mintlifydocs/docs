## üõ†Ô∏è Request Parameters

| Parameter       | Type    | Required | Description                                                                 |
|----------------|---------|----------|-----------------------------------------------------------------------------|
| `prompt`       | String  | Optional | A natural language prompt for music generation                              |
| `music_style`  | String  | Optional | Style of music to generate (e.g., Rock, Pop)                                |
| `lyrics`       | String  | Optional | Custom lyrics to be used in the music                                       |
| `voice_id`     | String  | Optional | Voice model to apply on generated music                                     |
| `webhook_url`  | String  | Optional | Webhook URL to receive a callback with the final audio result               |

---

## üéöÔ∏è Modes of Generation

### üéµ Custom Mode
Supply both `lyrics` and `music_style` to craft a personalized track with your exact preferences.

### ‚úçÔ∏è Prompt Mode
Just provide a `prompt`, and let the model generate everything for you ‚Äî from lyrics to melody and instrumentation.

> `voice_id` and `webhook_url` work in both modes for more control and automation.

## üì• Sample Request

### üêç Python
```python
import requests

url = "https://api.musicgpt.com/api/public/v1/MusicAI"
headers = {
    "Authorization": "<api_key>",
    "Content-Type": "application/json"
}
payload = {
    "prompt": "song about a football match between Man City and Arsenal",
    "music_style": "Pop",
    "webhook_url": "https://abcd.requestcatcher.com/",
    "lyrics": "",
    "voice_id": "Drake"
}

response = requests.post(url, json=payload, headers=headers)
print(response.json())
```
