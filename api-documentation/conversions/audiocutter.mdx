## ğŸ› ï¸ Request Parameters

| Parameter            | Type    | Required | Description                                                   |
|----------------------|---------|----------|---------------------------------------------------------------|
| `audio_path`         | String  | Yes      | The S3 path of the input audio file to trim                   |
| `conversion_id`      | String  | Yes      | A unique identifier for the conversion request               |
| `start_time`         | Float   | Yes      | The start time (in seconds) of the audio to trim              |
| `end_time`           | Float   | Yes      | The end time (in seconds) of the audio to trim                |
| `output_extension`   | String  | Yes      | Output format: mp3, wav, flac, ogg, aac, or webm              |

## ğŸ“¥ Sample Request

### ğŸ Python
```python
import requests

url = "https://api.musicgpt.com/api/public/v1/audio_cutter"
headers = {
    "accept": "application/json",
    "Authorization": "<api_key>",
    "Content-Type": "application/json"
}
payload = {
    "audio_path": "https://musicgpt.s3.us-east-1.amazonaws.com/conversions/sample-audio.mp3",
    "conversion_id": "user_trim_12345",
    "start_time": 10.5,
    "end_time": 45.0,
    "output_extension": "mp3"
}

response = requests.post(url, json=payload, headers=headers)
print(response.json())
```
