## 📖 Welcome to the **Lalals Public API Documentation**

This guide provides everything you need to integrate with our **audio generation and conversion services**, including music generation, extraction, remix, text-to-speech, and many more.

🎯 Whether you're building the next-gen AI music tool, automating voice workflows, or just experimenting with creative conversions — we've got you covered.

---

## ✨ Quickstart Guide

Get started in no time by following this simple checklist:

1. 🔑 **[Get your API Key](https://lalals.com/api)** to authorize your requests.
2. 🎼 **Make your conversion** by calling any of the [conversion endpoints](#🎼-conversion-features).
3. 💬 **Provide a webhook URL** to receive the output once the conversion is complete.
4. ⏳ **Track your conversion** using the [Check Conversion Status by ID](#📈-check-conversion-status-by-task-id) endpoint.
5. 🤔 **Need a specific voice?** Use the [Search Voice ID](#🔍-search-voice-id) or [Get All Voice IDs](#🧠-get-all-voice-ids) endpoints.

## 🔐 Authentication

All endpoints require an **API Key**.
Include it in the `Authorization` header like this:

```bash
Authorization: YOUR_API_KEY
```

## ⚠️ Rate Limits

To ensure fair API usage, the following rate limits apply:

- **MusicAI:** Maximum **10 parallel** audio creations.
- **Other Conversions:** Maximum **60 conversions per minute**.
- **GetConversionById:** Maximum **200 API calls per minute**.

> 💡 **Note:** Adhering to rate limits prevents throttling and API disruptions.
